Manual Técnico: Integración WebService REST/JSON

1. Autenticación y Seguridad

Para acceder a la API pública, es obligatorio incluir el Token de Seguridad Maestro en cada petición para validar la identidad de su plataforma.

M7-SECURE-2026-XQW

Métodos de Envío:

  • Encabezado (Header): x-public-token: M7-SECURE-2026-XQW
  • Parámetro URL (Query): ?token=M7-SECURE-2026-XQW

2. Consulta de Pedidos

El WebService expone un punto de enlace único para la descarga de información logística bajo tres modalidades de filtrado.

GET /api/grupo-inter/public/list

Modalidad 1: Rango de Fechas Personalizado

Obtenga documentos procesados en un periodo específico (Formato YYYY-MM-DD).

Ejemplo de Petición URL:

GET https://orbitm7.m7apps.com/api/grupo-inter/public/list?token=M7-SECURE-2026-XQW&fechaDesde=2026-03-01&fechaHasta=2026-03-07

Modalidad 2: Consulta por Número de Documento

Permite obtener directamente la información de un documento específico (Factura). Esta consulta no tiene restricción de fecha.

Ejemplo de Petición URL:

GET https://orbitm7.m7apps.com/api/grupo-inter/public/list?token=M7-SECURE-2026-XQW&nroDocumento=FI100020

Modalidad 3: Consulta Histórica (Últimos 8 Días)

Si se omiten los filtros anteriores, el sistema entrega automáticamente los registros de los últimos 8 días calendario.

Ejemplo de Petición URL:

GET https://orbitm7.m7apps.com/api/grupo-inter/public/list?token=M7-SECURE-2026-XQW
